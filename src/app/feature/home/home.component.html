ADN Procesos Etapas de Diversiones


Aplicativo WEB que consiste en realizar la compra de procesos para distintos etapas de diversiones. Se manejara el registro de etapas con datos como: código del etapa, nombre, dirección y teléfono. Se manejara un registro de clientes con datos como: nombre y cedula.

Reglas de negocio:

1- El servicio para la compra de procesos puede ser únicamente de Martes a Domingo, puesto que los lunes son asignados para realizar labores de mantenimiento del etapa.
2- De Martes a Jueves el proceso tiene un valor de $15.000.
3- Los fines de semana (viernes, sábado y domingo) el proceso tiene un valor del 50% adicional en comparación de los otros días.
4- El etapa de diversiones cuenta únicamente con 50 procesos por dia.
5- Cada persona puede adqurir un maximo de 5 procesos por dia con el mismo etapa.
